        4.install cors - npm i cors

        cors
        ----
        The word CORS stands for “Cross-Origin Resource Sharing”.
        Cross-Origin Resource Sharing is an HTTP-header based mechanism implemented by the
        browser which allows a server or an API(Application Programming Interface) to indicate 
        any origins (different in terms of protocol, hostname, or port) other than its origin from which
        the unknown origin gets permission to access and load resources.
        The cors package available in the npm registry is used to tackle CORS errors in a Node.js application.
